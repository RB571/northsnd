Northsnd – Albums & Playlists

Gebruik:
1) Open 'northsnd-albums.html' in je browser.
2) Klik bij elk album op 'Instellen' om een YouTube-playlist URL of ID toe te voegen.
3) Opslaan → de playlist wordt lokaal bewaard in je browser (localStorage).
4) Je hebt 8 albums beschikbaar, elk met eigen titel en beschrijving.

Opmerking:
- De styling gebruikt Tailwind via een CDN. Zonder internetverbinding werkt de layout mogelijk niet correct.
- Dit is een statische site, je kunt hem uploaden naar GitHub Pages, Netlify of een eigen server.
